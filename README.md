# Projeto consulta preços de produtos

## Com esse projeto você é capaz de consultar o preço dos produtos previamente antes de ir as compras

## Tecnologias Utilizadas

- **Java 17**
- **Spring Boot**
- **Maven**
- **Docker**
- **Selenium**
- **Redis Stream**
- **Docker compose**
